Document Overview

This guide provides the needed information to successfully deploy Cloudian HyperFile VM and covers the basics of installation and configuration. While the installation part covers specifically the VM deployment, the configuration part is applicable as well to the Appliance version (HFA-xxx). We focus on using GUI. For CLI and other management tools, refer to the product manual.

This guide doesn’t replace the HyperFile Administrator Guide or the product manuals that provide a product deeper dive.

This guide is organized in three main sections:

  • Solution overview and pre-requisites for installation (detailed information in the Appendix A)
  • Installation and basic configuration
  • Advanced configuration and disk devices managements (Appendix B)

A sample Test Plan is provided in Appendix B.
